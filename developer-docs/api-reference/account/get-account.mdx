---
title: "Get Account"
api: 'POST /get_account'
description: "Get your account details including name, plan, status, credit balance, and API key count."
---

## Connection

| | |
|---|---|
| **Method** | `POST` |
| **URL** | `https://api.moltsets.com/api/v1/tools/get_account` |
| **Auth** | `Authorization: Bearer YOUR_API_KEY` |
| **Content-Type** | `application/json` |

## Parameters

This tool takes no arguments. Send an empty JSON object `{}` as the request body.

## Request

```bash
curl -X POST https://api.moltsets.com/api/v1/tools/get_account \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{}'
```

## Response fields

| Field | Type | Description |
|---|---|---|
| `results.id` | string | Account ID |
| `results.name` | string | Account name |
| `results.status` | string | Account status — `active` or `suspended` |
| `results.plan` | string | Current plan — `free`, `starter`, `pro`, etc. |
| `results.credit_balance` | integer | Remaining credits |
| `results.unlimited` | boolean | `true` if the account has unlimited credits |
| `results.api_key_count` | integer | Number of API keys on the account |
| `results.user_count` | integer | Number of users on the account |
| `results.created_at` | string | ISO 8601 account creation timestamp |
| `metadata.tokens_charged` | integer | Credits consumed — always `0` for this tool |
| `metadata.tokens_remaining` | integer | Credits remaining after this request |
