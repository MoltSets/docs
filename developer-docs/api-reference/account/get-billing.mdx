---
title: "Get Billing"
api: 'POST /get_billing'
description: "Get your account's billing details including current plan, subscription status, credit balance, renewal date, and credit costs per tool."
---

## Connection

| | |
|---|---|
| **Method** | `POST` |
| **URL** | `https://api.moltsets.com/api/v1/tools/get_billing` |
| **Auth** | `Authorization: Bearer YOUR_API_KEY` |
| **Content-Type** | `application/json` |

## Parameters

This tool takes no arguments. Send an empty JSON object `{}` as the request body.

## Request

```bash
curl -X POST https://api.moltsets.com/api/v1/tools/get_billing \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{}'
```

## Response fields

| Field | Type | Description |
|---|---|---|
| `results.plan` | string | Current plan — `free`, `starter`, `pro`, etc. |
| `results.subscription_status` | string | Subscription state — `active`, `none`, `cancelled`, etc. |
| `results.credit_balance` | integer | Remaining credits |
| `results.unlimited` | boolean | `true` if the account has unlimited credits |
| `results.current_period_start` | string \| null | ISO 8601 start of the current billing period |
| `results.current_period_end` | string \| null | ISO 8601 end of the current billing period |
| `results.token_costs` | object | Credit cost per tool call — keys are tool names, values are integer credit costs |
| `metadata.tokens_charged` | integer | Credits consumed — always `0` for this tool |
| `metadata.tokens_remaining` | integer | Credits remaining after this request |
