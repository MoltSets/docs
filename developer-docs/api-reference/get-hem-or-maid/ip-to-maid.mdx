---
title: "IP to MAID"
api: 'POST /ip_to_maid'
description: "Resolve an IP address to mobile advertising IDs (MAIDs) associated with it."
---

## Connection

| | |
|---|---|
| **Method** | `POST` |
| **URL** | `https://api.moltsets.com/api/v1/tools/ip_to_maid` |
| **Auth** | `Authorization: Bearer YOUR_API_KEY` |
| **Content-Type** | `application/json` |

## Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `ip_address` | string | Yes | IPv4 address to resolve |

## Request

```bash
curl -X POST https://api.moltsets.com/api/v1/tools/ip_to_maid \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"ip_address": "8.8.8.8"}'
```

## Response fields

| Field | Type | Description |
|---|---|---|
| `results` | array | List of matched MAID objects |
| `results[].device_id` | string | Mobile advertising ID (IDFA or AAID) |
| `results[].device_type` | string | ID type — `AAID` (Android) or `IDFA` (iOS) |
| `metadata.tokens_charged` | integer | Credits consumed by this request |
| `metadata.tokens_remaining` | integer | Credits remaining after this request |

<Info>
  Credits are only charged when results are found. If no MAIDs are matched, `tokens_charged` will be `0` and `results` will be an empty array.
</Info>
