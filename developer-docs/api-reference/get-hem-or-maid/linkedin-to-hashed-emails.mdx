---
title: "LinkedIn to HEM"
api: 'POST /linkedin_to_hashed_emails'
description: "Look up all known hashed email addresses (MD5 and SHA-256) for a LinkedIn profile, across both business and personal emails."
---

## Connection

| | |
|---|---|
| **Method** | `POST` |
| **URL** | `https://api.moltsets.com/api/v1/tools/linkedin_to_hashed_emails` |
| **Auth** | `Authorization: Bearer YOUR_API_KEY` |
| **Content-Type** | `application/json` |

## Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `linkedin_url` | string | Yes | Full LinkedIn profile URL or slug (e.g. `https://linkedin.com/in/john-smith-123`) |

## Request

```bash
curl -X POST https://api.moltsets.com/api/v1/tools/linkedin_to_hashed_emails \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"linkedin_url": "https://linkedin.com/in/john-smith-123"}'
```

## Response fields

| Field | Type | Description |
|---|---|---|
| `results.linkedin_slug` | string | LinkedIn slug for the resolved profile |
| `results.business_md5_array` | array of strings | MD5 hashes of known business email addresses |
| `results.business_sha256_array` | array of strings | SHA-256 hashes of known business email addresses |
| `results.personal_md5_array` | array of strings | MD5 hashes of known personal email addresses |
| `results.personal_sha256_array` | array of strings | SHA-256 hashes of known personal email addresses |
| `metadata.tokens_charged` | integer | Credits consumed by this request |
| `metadata.tokens_remaining` | integer | Credits remaining after this request |

<Info>
  Credits are only charged when results are found. If no hashes are found, `tokens_charged` will be `0` and the arrays will be empty.
</Info>
