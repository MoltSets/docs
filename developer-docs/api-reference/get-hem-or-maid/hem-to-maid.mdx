---
title: "HEM to MAID"
api: 'POST /hem_to_maid'
description: "Resolve a hashed email (MD5) to mobile advertising IDs (MAIDs) associated with it."
---

## Connection

| | |
|---|---|
| **Method** | `POST` |
| **URL** | `https://api.moltsets.com/api/v1/tools/hem_to_maid` |
| **Auth** | `Authorization: Bearer YOUR_API_KEY` |
| **Content-Type** | `application/json` |

## Parameters

<Tip>
  Pass either `md5` or `email` — not both.
</Tip>

| Parameter | Type | Required | Description |
|---|---|---|---|
| `md5` | string | Conditional | MD5 hash of the email address |
| `email` | string | Conditional | Plaintext email address |

## Request

```bash
curl -X POST https://api.moltsets.com/api/v1/tools/hem_to_maid \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"md5": "5d41402abc4b2a76b9719d91101XXXXX"}'
```

## Response fields

| Field | Type | Description |
|---|---|---|
| `results` | array | List of matched MAID objects |
| `results[].device_id` | string | Mobile advertising ID (IDFA or AAID) |
| `results[].device_type` | string | ID type — `AAID` (Android) or `IDFA` (iOS) |
| `metadata.tokens_charged` | integer | Credits consumed by this request |
| `metadata.tokens_remaining` | integer | Credits remaining after this request |

<Info>
  Credits are only charged when results are found. If no MAIDs are found, `tokens_charged` will be `0` and `results` will be an empty array.
</Info>
