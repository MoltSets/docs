---
title: "LinkedIn to Mobile Phone"
api: 'POST /enrich_phone'
description: "Mobile Phone Enrichment — find the verified mobile number for a LinkedIn profile."
---

Only mobile numbers are returned — no landlines. Credits are only charged when data is found.

Supports single lookups via `linkedin_url` or batch lookups via `linkedin_urls` (up to 100). These parameters cannot be combined.

## Connection

| | |
|---|---|
| **Method** | `POST` |
| **URL** | `https://api.moltsets.com/api/v1/tools/enrich_phone` |
| **Auth** | `Authorization: Bearer YOUR_API_KEY` |
| **Content-Type** | `application/json` |

## Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `linkedin_url` | string | Conditional | Single LinkedIn profile URL (cannot combine with `linkedin_urls`) |
| `linkedin_urls` | array of strings | Conditional | Batch of LinkedIn profile URLs, up to 100 (cannot combine with `linkedin_url`) |

## Request

**Single lookup:**

```bash
curl -X POST https://api.moltsets.com/api/v1/tools/enrich_phone \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"linkedin_url": "https://linkedin.com/in/john-smith-123"}'
```

**Batch lookup:**

```bash
curl -X POST https://api.moltsets.com/api/v1/tools/enrich_phone \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"linkedin_urls": ["https://linkedin.com/in/john-smith-123", "https://linkedin.com/in/jane-doe-456"]}'
```

## Response fields

| Field | Type | Description |
|---|---|---|
| `results.mobile_phone` | string | Mobile phone number (single lookup) |
| `results` | array | Array of result objects for batch lookups, each with `linkedin_url` and `mobile_phone` |
| `metadata.tokens_charged` | integer | Credits consumed by this request (10 per result found) |
| `metadata.tokens_remaining` | integer | Credits remaining after this request |
