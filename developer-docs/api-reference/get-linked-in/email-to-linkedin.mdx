---
title: "Email to LinkedIn"
api: 'POST /hem_to_linkedin'
description: "Resolve a plaintext email address to a LinkedIn profile slug."
---

## Connection

| | |
|---|---|
| **Method** | `POST` |
| **URL** | `https://api.moltsets.com/api/v1/tools/hem_to_linkedin` |
| **Auth** | `Authorization: Bearer YOUR_API_KEY` |
| **Content-Type** | `application/json` |

## Parameters

<Tip>
  Pass either `email` or `md5` — not both.
</Tip>

| Parameter | Type | Required | Description |
|---|---|---|---|
| `email` | string | Conditional | Plaintext email address |
| `md5` | string | Conditional | MD5 hash of the email address |

## Request

```bash
curl -X POST https://api.moltsets.com/api/v1/tools/hem_to_linkedin \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"email": "john.smith@example.com"}'
```

## Response fields

| Field | Type | Description |
|---|---|---|
| `results.linkedin_slug` | string | LinkedIn profile slug (e.g. `john-smith-123`) |
| `metadata.tokens_charged` | integer | Credits consumed by this request |
| `metadata.tokens_remaining` | integer | Credits remaining after this request |

<Info>
  Credits are only charged when a result is found. If no match is found, `tokens_charged` will be `0` and `results` will be empty.
</Info>
