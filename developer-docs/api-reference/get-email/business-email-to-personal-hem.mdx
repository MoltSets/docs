---
title: "Business Email to Personal HEM"
api: 'POST /business_email_to_personal_hem'
description: "Convert a known business email address into a hashed personal email (HEM) for use in identity resolution and audience matching."
---

## Connection

| | |
|---|---|
| **Method** | `POST` |
| **URL** | `https://api.moltsets.com/api/v1/tools/business_email_to_personal_hem` |
| **Auth** | `Authorization: Bearer YOUR_API_KEY` |
| **Content-Type** | `application/json` |

## Parameters

<Tip>
  Pass either `email` or `md5` — not both.
</Tip>

| Parameter | Type | Required | Description |
|---|---|---|---|
| `email` | string | Conditional | Plaintext business email address |
| `md5` | string | Conditional | MD5 hash of the business email address |

## Request

```bash
curl -X POST https://api.moltsets.com/api/v1/tools/business_email_to_personal_hem \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"email": "john.smith@example.com"}'
```

## Response fields

| Field | Type | Description |
|---|---|---|
| `results.personal_email_sha256` | array of strings | SHA-256 hashes of associated personal email addresses |
| `metadata.tokens_charged` | integer | Credits consumed by this request |
| `metadata.tokens_remaining` | integer | Credits remaining after this request |

<Info>
  Credits are only charged when a result is found. If no match is found, `tokens_charged` will be `0` and `results` will be empty.
</Info>
