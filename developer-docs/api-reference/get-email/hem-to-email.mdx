---
title: "HEM to Email"
api: 'POST /hem_to_email'
description: "Resolve a hashed email (MD5) to the plaintext email address and SHA-256 hash."
---

## Connection

| | |
|---|---|
| **Method** | `POST` |
| **URL** | `https://api.moltsets.com/api/v1/tools/hem_to_email` |
| **Auth** | `Authorization: Bearer YOUR_API_KEY` |
| **Content-Type** | `application/json` |

## Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `md5` | string | Yes | MD5 hashed email address |

## Request

```bash
curl -X POST https://api.moltsets.com/api/v1/tools/hem_to_email \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"md5": "5d41402abc4b2a76b9719d91101XXXXX"}'
```

## Response fields

| Field | Type | Description |
|---|---|---|
| `results.email` | string | Resolved plaintext email address |
| `results.md5_email` | string | MD5 hash of the resolved email |
| `results.sha256_email` | string | SHA-256 hash of the resolved email |
| `metadata.tokens_charged` | integer | Credits consumed by this request |
| `metadata.tokens_remaining` | integer | Credits remaining after this request |

<Info>
  Credits are only charged when a result is found. If the hash cannot be resolved, `tokens_charged` will be `0` and `results` will be empty.
</Info>
