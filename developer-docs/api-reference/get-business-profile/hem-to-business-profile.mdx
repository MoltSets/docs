---
title: "HEM to Business Profile"
api: 'POST /hem_to_business_profile'
description: "Resolve a hashed email (MD5) to a full business profile including name, company, title, and contact info."
---

## Connection

| | |
|---|---|
| **Method** | `POST` |
| **URL** | `https://api.moltsets.com/api/v1/tools/hem_to_business_profile` |
| **Auth** | `Authorization: Bearer YOUR_API_KEY` |
| **Content-Type** | `application/json` |

## Parameters

<Tip>
  Pass either `md5` or `email` — not both. `include_mobile_phone` is optional and costs additional credits when data is found.
</Tip>

| Parameter | Type | Required | Description |
|---|---|---|---|
| `md5` | string | Conditional | MD5 hash of the email address |
| `email` | string | Conditional | Plaintext email address |
| `include_mobile_phone` | boolean | No | When `true`, attempts to include mobile phone in the response |

## Request

```bash
curl -X POST https://api.moltsets.com/api/v1/tools/hem_to_business_profile \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"md5": "5d41402abc4b2a76b9719d91101XXXXX", "include_mobile_phone": true}'
```

## Response fields

| Field | Type | Description |
|---|---|---|
| `results.first_name` | string | First name |
| `results.last_name` | string | Last name |
| `results.title` | string | Current job title |
| `results.seniority` | string | Seniority level (e.g. `Manager`, `Director`, `VP`) |
| `results.functional_area` | string | Department or functional area |
| `results.linkedinurl` | string | LinkedIn profile URL |
| `results.current_company` | string | Current employer name |
| `results.current_company_url` | string | Employer website URL |
| `results.current_company_linkedinurl` | string | Employer LinkedIn URL |
| `results.current_industry` | string | Industry classification |
| `results.country` | string | Country |
| `results.company_employee_count` | string | Approximate headcount |
| `results.company_employee_range` | string | Headcount range (e.g. `1-10`, `11-50`) |
| `results.company_revenue` | string | Estimated annual revenue in USD |
| `results.company_revenue_range` | string | Revenue range (e.g. `$1-10M`) |
| `results.work_email_confirmed` | string | MD5 hash of the confirmed work email |
| `results.work_email_confirmed_status` | string | Date the work email was last confirmed (`YYYY-MM-DD`) |
| `results.personal_emails` | array of strings | Known personal email hashes |
| `results.personal_emails_status` | array | Validation status for each personal email |
| `metadata.tokens_charged` | integer | Credits consumed by this request |
| `metadata.tokens_remaining` | integer | Credits remaining after this request |

<Info>
  Credits are only charged when a result is found. If no profile is matched, `tokens_charged` will be `0` and `results` will be empty.
</Info>
