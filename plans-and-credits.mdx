---
title: "Plans and credits"
---

moltsets uses a credit-based pricing model. A credit is consumed each time an API call returns data successfully — if a call returns no results, no credit is used.

## What is a credit?

One credit equals one successful API response. You're only charged for results, not attempts.

## Plans

| **Plan** | **Credits** | **Price** |
| :-- | :-- | :-- |
| Free | 250 credits / month | \$0 |
| Starter | 3,000 credits / month | \$149 / month |
| Growth | 10,000 credits / month | \$299 / month |
| Unlimited | Unlimited credits | \$499 / month |

## Choosing a plan

- **Free** is ideal for exploring the API and testing integrations.
- **Starter** suits low-volume agents or early production use.
- **Growth** is a good fit for teams running identity resolution at scale.
- **Unlimited** removes the cap entirely — best for high-throughput agents or production pipelines where volume is unpredictable.
